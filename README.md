# Advanced-Python-TextAnalysis-Project
## Assignment #2. Due December 8, 2025

Analyzing .txt/.pdf files of FDA Adverse Event Reports for kratom, 7-hydroxymitragynine, and mitragynine psuedoindoxyl. 

## Downloaded excel files from FDA FAERS system and analyze:
<ol>
1. The number of serious vs non-serious cases
2. The counts for each year (beginning in 2015)
3. The most common product names
4. The most common reason for use
5. The most common reactions
6. The average patient age
7. What reference literature is most commonly used for each case (where applicable)
</ol>

The analysis will look at data from the first reported case of kratom/kratom derivatives exposure through ~present day December 1, 2025. Herbals(mitragynine), 7-OH, and mitra pseduo will be compared together and against each other.
